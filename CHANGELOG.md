# Changelog

## 2026-06-29

Initial public repository package prepared for International Journal for Equity in Health
submission.

Included:

- cleaned aggregate country-level analysis dataset
- source indicator dictionary
- variable dictionary
- data visibility matrix
- main manuscript figures
- supplementary figures S1-S17
- supplementary tables S1-S20 as CSV
- manuscript result tables
- public reproduction script
- documentation for data sources, codebook, reproducibility, interpretation limits, and
  upload checks

Terminology was harmonised to avoid causal or ranking language. The repository uses
"measured exposure-side score" and "descriptive surveillance profiles" rather than
priority ranking terminology.
