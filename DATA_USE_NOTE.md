# Data use note

This repository provides cleaned aggregate country-level indicators assembled from public
global health and development sources. It does not contain individual-level data,
identifiable personal information, patient records, or restricted-access data.

## Source-provider terms

The cleaned files are derived from third-party public sources. Users should cite and
comply with the terms of the original providers, including:

- WHO Health Inequality Data Repository and Health Inequality Monitor
- WHO Global Health Observatory
- WHO Global Health Estimates
- IHME/GBD-linked estimates
- World Bank open indicators
- Global Data Lab HDI/SDI-related sources where applicable

The repository authors do not transfer ownership of the original source data. The MIT
License applies to the repository code and documentation, not to any third-party source
database.

## Appropriate reuse

The shared dataset is appropriate for reproducing the manuscript analyses, checking
country-level descriptive patterns, and developing transparent extensions of the
surveillance framework.

It is not appropriate for individual-level risk inference, clinical prediction, causal
attribution, or ranking countries as better or worse performers.

## Recommended citation practice

When using these files, cite:

1. The manuscript associated with this repository.
2. This repository or its archived DOI after release.
3. The original data providers relevant to the variables reused.
