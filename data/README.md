# Data folder

This folder contains cleaned aggregate country-level data and metadata.

| File | Description |
|---|---|
| `country_level_analysis_dataset.csv` | Main analytic dataset with one row per country |
| `variable_dictionary.csv` | Variable definitions and non-missing country counts |
| `source_indicator_dictionary.csv` | Source indicators, source files, units, year coverage, coverage counts, and notes |
| `data_visibility_matrix.csv` | Prevention-care pathway visibility classification |

The dataset contains no individual-level or identifiable data. See `../DATA_USE_NOTE.md`
and `../docs/INTERPRETATION_LIMITS.md` before reuse.
