# Women's lung-cancer prevention equity surveillance

This repository contains the public, aggregate, country-level data package and
reproducibility materials for the manuscript:

**Global visibility gaps in women's lung-cancer prevention equity:
a cross-national ecological study**.

The repository is designed for transparent review, reuse, and archiving. It includes the
cleaned analytic dataset, indicator dictionaries, pathway visibility assessment,
manuscript result tables, supplementary tables, supplementary figures, and a portable
script that reproduces key numerical checks and simplified figures.

## Study design and scope

This is a cross-national ecological study with one row per country. The analysis is a
prevention-equity surveillance and data-visibility assessment. It does not support
individual-level inference, clinical prediction, or causal attribution.

The central question is not whether any single exposure explains women's lung-cancer
burden globally. The central question is what current global data systems can and cannot
show about exposure-side inequities and prevention-care pathway visibility.

## Data sources

Source data were obtained from publicly available WHO HIDR-linked or compatible sources,
including WHO Health Inequality Data Repository, WHO Global Health Observatory, WHO Global
Health Estimates, IHME/GBD-linked estimates, World Bank indicators, and Global Data Lab.
Users should cite and comply with the terms of the original data providers.

See `docs/DATA_SOURCES.md` and `data/source_indicator_dictionary.csv` for source
provenance, year coverage, units, disaggregation, and modelling notes.

## Reproducibility

Install dependencies and run the public reproduction script from the repository root:

```bash
python -m pip install -r requirements.txt
python scripts/generate_figures.py
```

The script uses only relative paths. It writes reproduced numerical outputs to
`results/reproduced/` and simplified figure checks to `figures/reproduced/`.

The final manuscript-ready figures are provided in `figures/`. The reproduced figures are
intended as transparent checks rather than exact layout duplicates.

## Interpretation cautions

Smoking-adjusted burden residuals are descriptive country-level residuals after adjustment
for the available historical smoking proxy and structural covariates. They are not causal
estimates, not estimates of non-smoking lung cancer, and not direct environmental burden
estimates.

The historical smoking proxy uses female smoking prevalence in 2000, 2005, and 2010
because comparable earlier country-level female smoking data were not consistently
available. It should be interpreted as partial historical-smoking adjustment, not complete
lifetime tobacco exposure.

Measured exposure-burden profiles are descriptive surveillance profiles. They are not
priority rankings, national performance rankings, or validated risk categories.
