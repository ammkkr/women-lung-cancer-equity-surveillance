# Supplementary materials

This folder contains supplementary figures, supplementary tables, and indexes exported
from the manuscript supplementary appendix.

## Files

- `supplementary_figure_index.csv`: index of Figures S1-S17 with PNG and PDF file paths.
- `supplementary_table_index.csv`: index of Tables S1-S20 with CSV file paths, row counts,
  and column counts.
- `figures/`: Supplementary Figures S1-S17 in PNG and PDF.
- `tables/`: Supplementary Tables S1-S20 in CSV format.

## Interpretation note

Supplementary materials use the same interpretation boundaries as the main repository.
Smoking-adjusted burden residuals are descriptive country-level residuals and should not
be interpreted as causal environmental burden estimates. Measured exposure-burden profiles
are descriptive surveillance profiles, not rankings or validated risk categories.

## How to locate a supplement item

Start with the index files:

```text
supplementary/supplementary_figure_index.csv
supplementary/supplementary_table_index.csv
```

The CSV tables are provided so that reviewers and readers can inspect supplementary
results without relying on a Word document.
