# Results folder

This folder contains manuscript result tables and outputs regenerated by the public
reproduction script.

## Manuscript result tables

| File | Description |
|---|---|
| `clean_fuel_correlations.csv` | Spearman correlations linking clean-fuel deficit to development and rural-disadvantage indicators |
| `historical_smoking_model_coefficients.csv` | Historical smoking model coefficients across burden outcomes and sensitivity covariate sets |
| `historical_smoking_model_fit.csv` | Model fit statistics for historical smoking models |
| `pm25_robustness_results.csv` | PM2.5 residual robustness and sensitivity outputs |

## Reproduced outputs

Running:

```bash
python scripts/generate_figures.py
```

creates:

- `results/reproduced/reproduced_historical_smoking_models.csv`
- `results/reproduced/reproduced_pm25_residual_models.csv`
- `results/reproduced/reproduced_clean_fuel_correlations.csv`

These files are transparent checks based on the public repository data.
